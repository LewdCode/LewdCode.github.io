The following commands are currently registered with InteBot. All commands have a single colon as a prefix

  • :taco - Give a taco to a user
  • :pizza - Give a Pizza to a user
  • :flirt - Flirt with a user
  • :slap - Slap a user.. hard.
  • :kill - Worse than Slap
  • :cookie - Give a cookie to a user
  • :cake - Give a cake to a user
  • :420 - Blaze it.
  • :potato - Give a Potato to a user
