XonoxBot for Xonoxide - Command List

The following commands are currently registered with XonoxBot.

  • :taco - Give a taco to a user. Example :taco XonoxBot
  • :pizza - Give a Pizza to a user. Example :pizza XonoxBot
  • :flirt - Flirt with a user. Example :flirt XonoxBot
  • :slap - Slap a user.. hard. Example :slap XonoxBot
  • :kill - Worse than Slap. Example :kill XonoxBot
  • :cookie - Give a cookie to a user. Example :cookie XonoxBot
  • :cake - Give a cake to a user. Example :cake XonoxBot
  • :potato - Give a Potato to a user. Example :potato XonoxBot
  • :huggles - Give a user huggles. Example :huggles XonoxBot
  • :brownie - Give a user a delicious brownie! Example :brownie XonoxBot
  • :burger - Give a user a home crafted burger. Example :burger XonoxBot
  • :sandwich - Give a user a Sandwich
  • :shower - Get a shower thought directly to you.
  • :fight - Pit two people against each other.
  • :donate - Get XonoxBot to whisper the donation link to you. Donations to this link are for Supporting of XonoxBot development, and not the streamer.
  • :nacho - Give some nachos to a user
  • :burrito - Give someone a burrito!
  • :fortune - Get your fortune, direct from XonoxBot
  • :xonox - Check/Create a Xonochrome Account
  • :claim - Claim free Xonochromes
  • :gamble - Gamble your Xonochromes! Risks plus rewards!
  • :lart - LART - Luser Attitude Re-alignment Tool.
  • :get # - Get a Quote matching the #
  • 8ball ? - Ask the magic 8ball a question. No : needed. Must end in ?
